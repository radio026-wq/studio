# **App Name**: 026 Online Radio

## Core Features:

- Live Streaming: Stream live audio content to the online radio station.
- Contact Form: Enable users to contact the radio station via a contact form, including phone number (829-205-606) and email (dleon4@hotmail.com).
- Content Schedule: Display a schedule of upcoming shows and content.
- Automated Highlight Generation: AI Tool: Automatically create and display audio highlights to engage listeners and provide quick content previews using AI algorithms. This is a 'tool' to help extract audio segments automatically.

## Style Guidelines:

- Primary color: Deep violet (#7957D6) to symbolize creativity and originality.
- Background color: Light grayish-violet (#EEE9F8) for a calm and relaxed listening environment.
- Accent color: Light blue (#5DBEEA) to create a refreshing contrast and evoke feelings of trust and communication.
- Body and headline font: 'Inter' for a modern and clean look. 
- Use modern, minimalist icons for navigation and player controls.
- Clean, responsive layout that adapts to different screen sizes.
- Subtle transitions and animations for a smooth user experience.
